# Зовнішній редактор
Внутрішньоігровий текстовий редактор зазвичай достатній для гри, але, звісно, він не може конкурувати з більш складними редакторами, такими як Visual Studio Code.

Гра зберігає всі файли коду як `.py` файли, тому ви можете редагувати їх за допомогою редакторів Python.
Зауважте, що це лише для зручності. Мова гри насправді не є Python, але вона достатньо близька, щоб Python IntelliSense працював з нею коректно.
Ви можете знайти файли в [теці save](persistent_data_path/Saves).

Кожне збереження також містить файл `__builtins__.py`, який містить вбудовані визначення Python, що відповідають ігровим builtins, для роботи IntelliSense.
VS Code може виявити `__builtins__.py` автоматично, але деякі редактори можуть працювати лише якщо ви зробите `from __builtins__ import *`.

Щоб бачити зовнішні зміни у грі без необхідності перезавантажувати збереження, ви повинні увімкнути опцію "File Watcher". Якщо ви створюєте або видаляєте файли зовні, вам все одно доведеться перезавантажити збереження, щоб побачити зміни.

## Використання VS Code
Visual Studio Code — це рекомендований редактор коду для використання з *The Farmer Was Replaced*.

Ви можете встановити його [тут](https://code.visualstudio.com/download).

Після завантаження встановіть розширення Python у VS Code.

Коли все готово, відкрийте [теку](persistent_data_path/Saves), де знаходяться ваші `.py` файли, у VS Code. Переконайтеся, що ви відкриваєте **всю теку**, а не окремі файли, інакше файл `__builtins__.py` не працюватиме.

У грі переконайтеся, що опція "Відслідковування файлів" увімкнена. Тепер щоразу, коли ви зберігаєте код у VS Code, зміни автоматично з'являтимуться у грі.

Ось і все! Тепер ви можете писати свій код у професійному редакторі!
