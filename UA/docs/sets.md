# Sets (Множини)
Множини (Sets) схожі на [dictionaries](docs/scripting/dicts.md), але без значень. Це просто невпорядкований набір ключів.

Вони створюються як словники, але без значень.
`set = {North, East, West}`

Використовуйте `set()`, щоб створити порожню множину. Зауважте, що `{}` створює порожній словник.

Використовуйте `set.add(elem)`, щоб додати новий елемент до множини.

Використовуйте `set.remove(elem)`, щоб видалити елемент із множини.

Використовуйте `if elem in set:`, щоб перевірити, чи містить множина елемент.

Використовуйте `for elem in set:`, щоб ітерувати (перебрати) всі елементи в множині.
Для великих множин оператор `in` працює набагато швидше, ніж для списку.

Так само, як і словники, множини є невпорядкованими, тому немає жодних гарантій щодо порядку, в якому елементи ітеруються.

Крім того, елементи в множинах є унікальними, тому додавання елемента, який вже є в множині, не змінить множину.
