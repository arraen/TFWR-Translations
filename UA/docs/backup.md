# Loading Backups
На жаль, іноді файл збереження пошкоджується, або ви втрачаєте деякі файли коду. Якщо це сталося з вами, ви можете спробувати завантажити backup (резервну копію). Якщо це трапляється регулярно, спробуйте вимкнути Steam Cloud.

Backup створюється щоразу, коли гру збережено, і зберігається невелика кількість копій на випадок, якщо вам потрібно щось відновити.
Ці backups можна знайти в [папці backup](persistent_data_path/Backup). Вони є копіями збережень з [папки save](persistent_data_path/Saves).
Найпростіший спосіб завантажити backup — скопіювати папку потрібної версії до директорії save.

Збереження — це папка з файлом `save.json` та набором `.py` файлів.
Якщо ви втратили лише кілька файлів коду, або файли коду на місці, але файл `save.json` пошкоджено, ви також можете замінити лише пошкоджені частини відповідними файлами з папки backup.
