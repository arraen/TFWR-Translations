# Оператори
арифметичні оператори: `+, -, *, /, //, %, **`
оператори порівняння: `==, !=, <=, >=, <, >`
булеві оператори: `not, and, or`

Примітка: Усі числа в грі є числами з плаваючою комою. Тому всі арифметичні оператори є операторами з плаваючою комою.
`//` визначено так, щоб просто округлювати число вниз після ділення.

Для операторів присвоєння вам потрібно розблокувати технологію "Variables".

## Вступ
Оператори дозволяють порівнювати, змінювати та комбінувати значення.
Арифметичні оператори `+, -, *, /, //, %, **` використовуються для виконання загальних математичних операцій над числами.
Оператори порівняння `==, !=, <=, >=, <, >` використовуються для порівняння значень. Результат завжди `True` або `False`.
Логічні оператори (також звані булевими операторами) `not, and, or` використовуються для комбінування значень істинності.

## Арифметичні оператори
`+` та `-` використовуються для додавання та віднімання.

`2 + 3` дорівнює `5`
`3 - 2` дорівнює `1`

`*`, `/` та `//` використовуються для множення та ділення.

`2 * 3` дорівнює `6`
`5 / 2` дорівнює `2.5`

`//` робить те саме, що й `/`, але результат округлюється вниз до найближчого цілого числа.

`5 // 2` дорівнює `2`

`%` — це оператор за модулем , також відомий як оператор остачі. Він по суті ділить два числа, а потім повертає остачу. Ви також можете думати про це як про багаторазове віднімання правого числа від лівого, доки остача не стане меншою за праве число.

`4 % 2` дорівнює `0`
`5 % 2` дорівнює `1`
`6 % 2` дорівнює `0`
`2 % 6` дорівнює `2`
`1.5 % 1` дорівнює `0.5`

`**` — це оператор степеня.

`2**2` дорівнює `4`
`(-5)**3` дорівнює `-125`

## Оператори порівняння
`==` та `!=` використовуються для перевірки, чи є два значення "рівними" (`==`) або "нерівними" (`!=`). Їх можна використовувати для всіх типів значень.

`2 == 2` дорівнює `True`
`Entities.Bush != Entities.Bush` дорівнює `False`
`3 != 3 + 1` дорівнює `True`

`<=, >=, <, >` можна використовувати лише з числами. Вони перевіряють, чи є ліве число "меншим або рівним" (`<=`), "більшим або рівним" (`>=`), "меншим" (`<`) або "більшим" (`>`) за праве число.

`1 <= 1` дорівнює `True`
`2 >= 3` дорівнює `False`
`-2 < -1` дорівнює `True`
`6 > 6` дорівнює `False`

## Логічні оператори
`not` просто інвертує значення:

`not False` дорівнює `True`
`not True` дорівнює `False`

`and` повертає `True`, тільки якщо обидва значення є `True`

`True and True` дорівнює `True`
`True and False` дорівнює `False`
`False and False` дорівнює `False`

`or` повертає `True`, якщо хоча б одне зі значень є `True`

`True or True` дорівнює `True`
`True or False` дорівнює `True`
`False or False` дорівнює `False`
