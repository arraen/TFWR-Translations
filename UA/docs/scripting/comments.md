# Comments (Коментарі)
Коментарі — це частини коду, які ігноруються під час виконання (execution).
Коментарі можна додавати за допомогою `#`. Усе, що знаходиться в тому ж рядку після `#`, є коментарем і буде проігноровано.

`#this is a comment`

Це може бути корисно для додавання нотаток до коду, а також для тимчасового вимкнення частин коду без їх видалення.

Будь-який коментар у рядку перед визначенням функції (function definition) буде частиною спливаючої інформації, яка з'являється при наведенні курсору миші на назву функції.
`#This function does nothing.
def f():
    pass`
