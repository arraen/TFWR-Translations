# Комантарі
У вкладених циклах `break` завжди виходить з найвнутрішнішого циклу.

`#це коментар`

Це може бути корисно для додавання приміток до коду, а також для тимчасового вимкнення частин коду без їх видалення.

Будь-який коментар у рядку перед визначенням функції буде частиною спливаючої інформації, яка з'являється при наведенні курсору миші на ім'я функції.
`#Ця функція нічого не робить.
def f():
    pass`
