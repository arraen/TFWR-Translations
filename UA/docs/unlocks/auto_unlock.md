# Авто-розблокування
Щоб повністю автоматизувати гру, ви можете використовувати функцію `unlock()`, щоб автоматично розблоковувати функції.
Наприклад, ви можете використовувати `unlock(Unlocks.Speed)` та `unlock(Unlocks.Expand)`, щоб розблокувати функції швидкості та розширення.

Щоб визначити вартість розблокування, просто використовуйте функцію `get_cost()`, як для рослини або предмета.
Приклад:
`get_cost(Unlocks.Loops)`
повертає `{Items.Hay:5}`
