# Pumpkins (Гарбузи)
[Pumpkins](objects/pumpkin) (гарбузи) ростуть як морква на зораному ґрунті (tilled soil). Їх посадка коштує моркви.

Коли всі гарбузи в квадраті повністю виростуть, вони зростуться разом, щоб утворити гігантський гарбуз. На жаль, гарбузи мають 20% шанс загинути, коли вони повністю виростуть, тому вам доведеться пересаджувати мертві, якщо ви хочете, щоб вони об'єдналися.

Коли гарбуз гине, він залишає після себе мертвий гарбуз (dead pumpkin), який нічого не дасть при зборі. Посадка нової рослини на його місці автоматично видаляє мертвий гарбуз, тому немає потреби збирати його. `can_harvest()` завжди повертає `False` на мертвих гарбузах.

Урожайність гігантського гарбуза залежить від розміру гарбуза.

Гарбуз 1x1 дає `1*1*1 = 1` гарбуз.
Гарбуз 2x2 дає `2*2*2 = 8` гарбузів замість `4`.
Гарбуз 3x3 дає `3*3*3 = 27` гарбузів замість `9`.
Гарбуз 4x4 дає `4*4*4 = 64` гарбузів замість `16`.
Гарбуз 5x5 дає `5*5*5 = 125` гарбузів замість `25`.
Гарбуз `n`x`n` дає `n*n*6` гарбузів для `n >= 6`.

Гарна ідея — отримати гарбузи розміром принаймні 6x6, щоб отримати повний множник.

Це означає, що навіть якщо ви посадите гарбуз на кожній клітинці квадрата, один із гарбузів може загинути і перешкодити зростанню мегагарбуза.
