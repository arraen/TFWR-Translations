# Variables (Змінні)
Змінні можна розглядати як іменовані контейнери, які можуть містити значення.
Оператор `=` використовується для оголошення змінної та збереження в ній значення.

`variable_name = value`

Ліва частина (Left hand side) оператора — це ім'я змінної. Ви можете дати їй будь-яке ім'я.
Права частина (Right hand side) — це вираз, результуюче значення якого буде збережено в змінній.

Оголосити змінну з іменем `a` та зберегти в ній значення `5`:
`a = 5`
Оголосити змінну з іменем `b` та зберегти в ній значення, яке повертає `can_harvest()`:
`b = can_harvest()`

Не плутайте оператор `=` з оператором `==`.
Оператор `==` перевіряє, чи рівні два значення, і повертає `True` або `False`.
Оператор `=` присвоює значення справа імені зліва.

Після того, як змінній було присвоєно значення, ви можете використовувати її в коді, щоб отримати значення, яке вона містить.

`a = 5
for i in range(a):
	do_a_flip()`

Наведений вище цикл виконується 5 разів, оскільки `a` встановлено на `5`.
`i` в циклі `for` — це також змінна, якій автоматично присвоюється поточне значення послідовності на кожній ітерації циклу. (Її не обов'язково називати `i`, ви можете дати їй будь-яке допустиме ім'я змінної.)

Змінні також дозволяють робити те саме з циклом while:

`a = 5
i = 0
while i < a:
	do_a_flip()
	i = i + 1`

Це робить те саме, що й цикл for вище. Ми просто повинні збільшувати i вручну.
Зауважте, що для збільшення i ми встановлюємо його як його власне значення плюс `1`. Зміна значення змінної на основі її попереднього значення є дуже поширеним явищем.
Це можна скоротити за допомогою таких операторів: `+=, -=, *=, /=, %=`

`i = i + 1` це те саме, що `i += 1`
`a = a / 3` це те саме, що `a /= 3`
