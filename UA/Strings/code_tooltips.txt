@code_tooltip_set_execution_speed = `set_execution_speed(speed)`
Обмежує швидкість, з якою виконується програма, щоб краще бачити, що відбувається.

`speed` рівна `1` — це швидкість, яку дрон має без жодних покращень швидкості.
`speed` рівна `8` змушує код виконуватися у `8` разів швидше і відповідає швидкості дрона після `3` покращень швидкості.
`speed` рівна `0.5` змушує код виконуватися з половиною швидкості без покращень. Це може бути корисно, щоб побачити, що робить код.

Якщо `speed` швидша, ніж execution може зараз йти, він просто піде з максимальною швидкістю.

Якщо `speed` рівна `0` або від'ємна, швидкість змінюється назад на максимальну.
Ефект також припиниться, коли execution зупиниться.

повертає `None`

займає `200` тіків для виконання.

example:
`set_execution_speed(1)`

`min(a,b,c)`: Повертає мінімум із `a`, `b` та `c`.
`min(sequence)`: Повертає мінімум з усіх значень у послідовності.

займає #comparisons тіків для виконання.

example:
`min([3,6,34,16])`
@code_tooltip_abs = `abs(number)`
Обчислює абсолютне значення (модуль) числа.

повертає `number`, якщо `number` позитивне, `-number` в іншому випадку.

займає 1 тік для виконання.

example:
`abs(-69)`
@code_tooltip_random = `random()`
Вибирає випадкове число від 0 (включно) до 1 (виключно).

повертає випадкове число.

займає `1` тік для виконання.

example:
`def random_elem(list):
	index = random() * len(list) // 1
	return list[index]`
@code_tooltip_append = `list.append(element)` 
Додає `element` у кінець `list`.

повертає `None`

займає `1` тік для виконання.

example:
`list = []
list.append(1)`
@code_tooltip_add = `set.add(element)` 
Додає `element` до `set`.

повертає `None`

займає `element size` тіків для виконання.

example:
`set = {0}
set.add(1)`
@code_tooltip_remove = `collection.remove(element)` 
Видаляє перше входження `element` із `collection`.

повертає `None`

займає `element size` тіків для виконання на set, #comparisions + #shifts тіків на list.

example:
`list = [True, False, None]
list.remove(False)`
@code_tooltip_pop = `collection.pop()` 
Видаляє останній елемент зі списку (list) або вказаний елемент зі словника (dictionary).
`list.pop(i)` видаляє елемент за індексом `i` з `list`.

повертає видалений елемент

займає `key size` тіків для виконання на dictionary, `len(list) - i + 1` тіків на list.

example:
`list = [True, False, None]
list.pop(0)`
@code_tooltip_insert = `list.insert(i, element)` 
Вставляє `element` у `list` за індексом `i`.

повертає `None`

займає `len(list) - i + 1` тіків.

example:
`list = [1,2]
list.insert(0, 0)`
@code_tooltip_Items = Містить усі items, які можуть бути в інвентарі. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Entities = Містить усі типи рослин (plant types). Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Grounds = Містить усі можливі типи землі (ground types). Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Unlocks = Містить усі unlocks та upgrades у меню досліджень. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Hats = Містить усі типи капелюхів. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Leaderboards = Містить усі категорії таблиць лідерів. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_for = Цикл (loop), який ітерує по всіх елементах послідовності. Деякі мови програмування називають це циклом "foreach".
@code_tooltip_while = Зациклює виконання, доки умова не стане хибною (`False`).
@code_tooltip_def = Визначає функцію (function).
@code_tooltip_True = Булеве значення, яке завжди є істинним (true).
@code_tooltip_False = Булеве значення, яке завжди є хибним (false).
@code_tooltip_if = Виконує код, якщо умова є істинною (`True`).
@code_tooltip_else = Виконує код, якщо попередня умова `if` була хибною (`False`).
@code_tooltip_elif = Робить те саме, що й:
`else:
    if condition:`
@code_tooltip_None = Значення, що представляє відсутність значення.
@code_tooltip_continue = Негайно продовжує з наступною ітерацією циклу (loop iteration). Якщо є вкладені цикли, це завжди впливатиме на найглибший цикл.
@code_tooltip_break = Виривається (Break) з циклу і продовжує виконання інструкцій після циклу. Якщо є вкладені цикли, це завжди впливатиме на найглибший цикл.
@code_tooltip_North = Напрямок вгору на екрані. Якщо ви не перевернете екран.
@code_tooltip_East = Напрямок праворуч на екрані. Якщо ви не перевернете екран.
@code_tooltip_South = Напрямок вниз на екрані. Якщо ви не перевернете екран.
@code_tooltip_West = Напрямок ліворуч на екрані. Якщо ви не перевернете екран.
@code_tooltip_not = `not True` є `False`, а `not False` є `True`.
@code_tooltip_and = Оцінює перший операнд. Якщо він хибний (`False`, `0` і порожні колекції), повертає це значення негайно (short-circuiting), інакше оцінює і повертає другий операнд.
@code_tooltip_or = Оцінює перший операнд. Якщо він істинний (будь-що, крім `False`, `0` і порожніх колекцій), повертає це значення негайно (short-circuiting), інакше оцінює і повертає другий операнд.
@code_tooltip_return = Використовується для повернення (return) значення з функції.
@code_tooltip_pass = Нічого не робить. Може бути корисним, оскільки порожні блоки коду не допускаються.
