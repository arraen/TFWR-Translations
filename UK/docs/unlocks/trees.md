# Дерева
[Дерева](objects/tree) — це кращий спосіб отримати дерево, ніж кущі. Вони дають по 5 wood кожне. Як і кущі, їх можна садити на траві або ґрунті.

Дерева люблять простір, і посадка їх впритул одне до одного сповільнить їхній ріст. Час росту подвоюється для кожного дерева, що знаходиться на клітинці безпосередньо на півночі, сході, заході або півдні від нього. Отже, якщо ви посадите дерева на кожній клітинці, вони ростимуть у `2*2*2*2 = 16` разів довше.

<spoiler=show> Тут може стати в пригоді оператор `%`. Пам'ятайте, що оператор `%` повертає остачу від ділення. Парні числа при діленні на `2` мають остачу `0`, а непарні числа при діленні на `2` мають остачу `1`.
Отже, ви можете перевірити, чи є число парним, ось так:

`def is_even(n):
	return n % 2 == 0`

Це повертає `True`, якщо n парне, і `False`, якщо ні.
</spoiler>
