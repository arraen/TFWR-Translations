# Вартості
Будь-яку вартість можна представити як словник, що зіставляє предмети з числами.

Функція `get_cost()` повертає такий словник. Вона повертає вартість рослини або розблокування.

`get_cost(Entities.Pumpkin)`
повертає `{Items.Carrot:1}`

Для unlocks можна передати необов'язковий другий аргумент для рівня розблокування, вартість якого ви хочете отримати. За замовчуванням це поточний рівень розблокування.

`get_cost(Unlocks.Loops, 0)`
повертає `{Items.Hay:5}`

Для unlocks, які вже знаходяться на максимальному рівні, `get_cost()` поверне `None`.

Це можна використовувати так:
`cost = get_cost(something)
for item in cost:
	amount_of_this_item_needed = cost[item]`
