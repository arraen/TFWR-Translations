# Списки
Списки — це простий спосіб зберігати кілька значень в одній змінній.
Ви можете створювати нові списки ось так:

`list = [2, True, Items.Hay]`

Тепер список містить значення `2`, `True` та `Items.Hay`.
Список також може бути порожнім:

`empty_list = []`

Ви можете отримати доступ до елемента списку за його індексом. Індекс дорівнює `0` для першого елемента, `1` для другого елемента, `2` для третього...

садить моркву
`list = [Entities.Tree, Entities.Carrot, Entities.Pumpkin]
plant(list[1])`

Ви можете перебирати список за допомогою циклу for. Наступний приклад підсумовує всі елементи у списку.

`list = [4, 7, 2, 5]
sum = 0
for number in list:
	sum += number`
`sum` тепер дорівнює `18`

Наступні методи списку дозволяють додавати та видаляти елементи:

`list.append(elem)` додає елемент у кінець списку:

`list = [2, 6, 12]
list.append(7)`
`list` тепер `[2, 6, 12, 7]`

`list.remove(elem)` видаляє перше входження елемента зі списку:

`list = [1, 2, 4, 2]
list.remove(2)`
`list` тепер `[1, 4, 2]`

`list.insert(index, elem)` вставляє елемент за вказаним індексом:

`list = [Entities.Tree, Items.Hay]
list.insert(1, Items.Wood)`
`list` тепер `[Entities.Tree, Items.Wood, Items.Hay]`

`list.pop(index)` видаляє елемент за вказаним індексом.
Якщо індекс не вказано, видаляється останній елемент.

`list = [3, 5, 8, 25]
list.pop()`
`list` тепер `[3, 5, 8]`
`list.pop(1)`
`list` тепер `[3, 8]`

Функція `len()` повертає довжину списку.
`list = [3, 2, 1]
x = len(list)`
`x` тепер `3`

Списки мають семантику посилань. Це означає, що присвоєння списку змінній присвоює цій змінній той самий об'єкт списку, а не створює копію списку.
Якщо дві змінні посилаються на той самий список, зміни в списку будуть видні в обох.

`a = [1,2]
b = a
b.pop()`
`a` і `b` тепер обидва `[1]`
