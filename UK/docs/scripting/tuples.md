# Кортежі
Кортежі — це чудовий спосіб об'єднати кілька значень в одне значення.
Щоб створити кортеж, просто розділіть значення комами:

`tuple = 1, 2`

Ви також можете розпакувати їх назад у кілька змінних. У коді нижче кортеж `(1,2)` розпаковується у дві змінні `a` та `b`.

`a, b = 1, 2`

Кортежі можна індексувати як списки, але вони є незмінними і не можуть бути змінені після створення.

`tuple = 1, 2`

`print(tuple[1])`
друкує `2`

`tuple[0] = 3`
видає помилку
<unlock=dicts>
На відміну від списків, кортежі можна використовувати як ключі в словниках.

`d = {(1,2):(4,5)}

print(d[(1,2)])`
`prints` (4,5)</unlock>

Вони також можуть бути корисними для повернення кількох значень із функції.

`def f():
    return 1, 2

a, b = f()`
