# Зовнішній редактор
Вбудований в гру текстовий редактор зазвичай достатній для гри, але, звичайно, він не може конкурувати з більш досконалими текстовими редакторами, такими як Visual Studio Code.

Гра зберігає всі файли коду як файли .py, тому ви можете редагувати їх за допомогою редакторів Python. 
Зверніть увагу, що це зроблено виключно для зручності. Мова гри насправді не є Python, але вона настільки близька до неї, що Python IntelliSense працює на ній досить добре.
Ви можете знайти файли в [папці збереження](persistent_data_path/Saves).

Кожен файл збереження також містить файл `__builtins__.py`, який містить вбудовані визначення Python, що відповідають вбудованим функціям у грі, щоб увімкнути IntelliSense.
VS Code може автоматично виявити файл `__builtins__.py`, але деякі редактори можуть працювати лише в тому випадку, якщо ви виконаєте команду `from __builtins__ import *`.

Щоб побачити зовнішні зміни в грі без необхідності перезавантаження збереження, необхідно ввімкнути опцію «Монітор файлів». Якщо ви створюєте або видаляєте файли зовні, вам все одно доведеться перезавантажити збереження, щоб їх побачити.

## Використання VS Code
Visual Studio Code — це рекомендований редактор коду для використання з The Farmer Was Replaced.

Ви можете встановити його [тут](https://code.visualstudio.com/download).

Після завантаження встановіть розширення Python у VS Code.

Після цього відкрийте [папку](persistent_data_path/Saves), яка містить ваші файли `.py`, у VS Code. Переконайтеся, що ви відкрили всю папку, а не окремі файли, інакше файл `__builtins__.py` не працюватиме.

У грі переконайтеся, що опція «Монітор файлів» увімкнена. Тепер щоразу, коли ви зберігаєте файл у VS Code, зміни автоматично відображатимуться в грі.

Ось і все! Тепер ви можете писати код у професійному редакторі коду!
